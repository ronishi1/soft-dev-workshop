Grid

Bootstrap’s grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive.

Bootstrap Website
Grid is a key feature of Bootstrap used to create responsive websites that scale on all screen sizes. The screen is split into 12 columns, and you can set how many of those columns something will take at certain sizes. For instance, if you have 3 cards like below, you will likely want to only show one card taking up all 12 columns on mobile or else it would try to squish 3 cards in a tiny window. Take a look at the examples below to see how it works!
Grid documentation


Example 1 of Grid
Right now, on a full screen, all three cards will be on the same row (each card 4 columns). But if you scale it down, it will show two cards per row (each card 6 columns), then once small enough one card per row (each card 12 columns).
