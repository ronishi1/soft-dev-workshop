About Foundation

Setup

One way to use the Foundation framework is to add the CDN link into your HTML head section.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">
Another way is to download the complete version here for all the features available in the framework.
